## Image Guidelines

### Quality
- High resolution (minimum 1200px width for hero)
- Professional photography
- Well-lit, clear shots
- Show completed work and quality craftsmanship

### Style
- Consistent style across all images
- Natural lighting preferred
- Show the work in context (houses, buildings)
- Cardiff/Bristol area imagery where possible

### Format
- JPG format recommended for photos
- PNG for logos or graphics with transparency
- Optimize file sizes (aim for under 500KB per image)
- Use tools like TinyPNG or ImageOptim to compress

### Optimization Before Upload
1. Resize to appropriate dimensions
2. Compress to reduce file size
3. Convert to web-friendly format
4. Test loading speed
