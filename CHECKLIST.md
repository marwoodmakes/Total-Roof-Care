## üÜò Common Issues & Solutions

### Issue: Images Not Displaying
**Solution:** Check file names match exactly (case-sensitive) and files are in /images/ folder

### Issue: Contact Form Not Sending
**Solution:** Set up Formspree or EmailJS (see QUICK-START.md)

### Issue: Mobile Menu Not Working
**Solution:** Verify js/script.js is loading, check browser console for errors

### Issue: Map Not Showing
**Solution:** Replace Google Maps iframe with your own embed code

### Issue: Slow Loading
**Solution:** Compress images using [TinyPNG](https://tinypng.com/), optimize file sizes

### Issue: SEO Not Working
**Solution:** Add structured data, meta descriptions, and submit sitemap to Google
